# HUB ASTORIE APP v1.4.9 – Admin Kontakty Role UX SAFE

Bezpečný dílčí release zaměřený pouze na administraci kontaktů partnerů.

## Změněno
- Admin → Kontakty: nový pracovní přehled s kompaktním zobrazením, rychlými filtry a sticky pracovním panelem.
- Partner/pojišťovna se vybírá přímo ze seznamu existujících partnerů.
- Přidán číselník rolí kontaktů: Admin → Role kontaktů.
- Role se řadí do skupin pro budoucí poradenské zobrazování: VIP, Infolinka, Smlouvy, Likvidace, Provize, Ostatní.
- Lze vybrat přednastavenou roli nebo zapsat jinou, která se uloží do číselníku.

## Beze změny
- Nový TIP
- Moje TIPy
- Partneři
- Sazebník
- Výpovědi
- E-mail core
- Datový model TIPů
