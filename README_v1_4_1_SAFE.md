# v1.4.1 TIP Workflow Real Operations SAFE

Tento release opravuje provozní práci s TIPy po jejich založení.

## Opravy
- Specialistovi lze v detailu TIPu doplnit:
  - stav TIPu,
  - číslo smlouvy,
  - datum uzavření,
  - výši obchodu / pojistné,
  - další obchod,
  - průběžnou informaci pro tipaře,
  - interní poznámku,
  - konečnou zprávu.
- Centrální evidence TIPů v administraci je kompaktnější.
- Stavy TIPů mají odlišné barvy.
- Moje TIPy mají rozšířený přehled provozních údajů.

## Bezpečnost
Release nemění Partnery ani Sazebník. Neprovádí destruktivní migrace.
